# Agent: Backend & Cloud Security

## Tech Stack Trigger

Load this agent when working with: Vercel Functions, Supabase Auth, or Cloud Enclaves.

## Current Status: DEFERRED

The project is currently in **Offline-First Mode**. No active cloud implementation is allowed unless authorized by the Orchestrator.

## Future Protocol

- **Zero-Knowledge:** The server must never see raw data or Master Passwords.
- **Portability:** Use Vercel Blob for encrypted audio and Postgres for encrypted text blobs.
